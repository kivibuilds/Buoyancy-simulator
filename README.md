# 🌊 Interactive Buoyancy Simulator

An interactive physics simulation that visualizes **Archimedes' Principle** by demonstrating how buoyant force, gravity, density, and displaced fluid determine whether an object floats, sinks, or achieves neutral buoyancy.

Designed as an educational web application, the simulator allows users to experiment with different object properties and liquid densities while observing real-time physics calculations and animations.

---

## 🚀 Features

- 🌊 Real-time buoyancy simulation
- ⚖️ Adjustable object mass and volume
- 🧪 Multiple liquid types
  - Water
  - Oil
  - Salt Water
  - Mercury
- 🔷 Multiple object shapes
- 📊 Live density calculations
- 📐 Real-time buoyant & gravitational force calculations
- 🌊 Interactive Canvas-based animations
- 💧 Dynamic waves, bubbles, and splash effects
- 📚 Educational explanation of Archimedes' Principle
- 📱 Responsive design for desktop and mobile

## 🧠 Physics Concepts

This simulator demonstrates:

- Archimedes' Principle
- Buoyant Force
- Density
- Gravitational Force
- Fluid Displacement
- Neutral Buoyancy

### Density

```
Density = Mass / Volume
```

### Buoyant Force

```
Buoyant Force = Liquid Density × Displaced Volume × Gravity
```

The simulator continuously compares:

- Object Density
- Liquid Density
- Buoyant Force
- Gravitational Force

to determine whether the object floats or sinks.

## 🎮 How to Use

1. Adjust the object's **mass** and **volume**.
2. Select a liquid.
3. Choose an object shape.
4. Click **Drop Object**.
5. Observe the simulation.
6. Compare the live physics values and force calculations.
